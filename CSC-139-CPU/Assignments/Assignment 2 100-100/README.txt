Before we begin:
	Make sure you have CPUSchedulingSimulation.c & input.1 in your working directory.

To Compile:
	type: gcc -g CPUSchedulingSimulation.c -o schedule

To Run:
	type for First Come First Serve:        ./schedule input.1 FCFS
	     for Round Robbin          :        ./schedule input.1 RR
	     for Shortest Remaining Time First: ./schedule input.1 SRTF
